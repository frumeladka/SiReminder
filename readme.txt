SiReminder

Инструкция по запуску:
1. В телеграме при помощи @BotFather создать бота и скопировать токен
2. Скачать и распаковать архив с проектом
3. Открыть (как текстовый документ) config.py и записать токен, полученный ранее, в переменную TOKEN
4. Скачать и установить Python 3.13
5. Скачать библиотеки из requirements.txt (в командой строке перейти в каталог с проектом и ввести команду "pip install -r requirements.txt")
6. Запустить main.py
